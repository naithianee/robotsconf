# Storage Phx
Initiated to facilitate migration between  applications and pplnix implementations.

Built for specific requirements, initially supporting read operations and thumbnail generation from local filesystem and Cloudflare R2. Direct uploads and additional providers might be added later, though not currently prioritized.

## Status
Active development phase. Deploy with caution until interface stabilizes.
